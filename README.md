# 3 Card Night Mode

A clean, responsive card UI built with vanilla HTML, CSS, and JavaScript. Features a dark mode toggle that persists across page refreshes.

## Features

- 3 styled cards with titles, descriptions, and labels
- White and purple color theme
- Dark mode toggle in the header with smooth transitions
- Theme preference saved to `localStorage`

## File Structure

```
├── index.html   # Markup
├── style.css    # CSS variables, dark/light themes, component styles
└── app.js       # Dark mode toggle logic
```

## Usage

Open `index.html` in a browser. No build step or dependencies required.
